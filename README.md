## This is project implemented based on Next.js and Mapbox

### Tech stack

- react
- nextjs
- Mapbox
- typescript
- jest
- cypress

![](./public//images/demo.jpg)

## How to run it?

```sh
// client side
npm i
npm run dev

// local server if you want to test it
cd server
npm i
npm run dev:serve
```

## Run the test

```sh
// cypress
npm run test:cypress

// jest
npm run test:jest
```
